# mouseroad-wetty-server

這是一個本地端的 Web Bash (支援 git) 伺服器，使用flask。

## 安裝步驟

1. 安裝 flask
2. 在專案目錄下執行：
   ```sh
   pip install flask
   ```

## 啟動伺服器

```sh
app.py
```

啟動後，瀏覽器打開 [127.0.0.1:5000](127.0.0.1:5000) 即可使用 bash（支援 git 指令）。
